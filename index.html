Projeto desenvolvido para a disciplina de Teste de Software da Católica SC. Hospedagem estática via Cloudflare Pages conectada ao front e backend no Render.com e Com banco de dados no Turso

Acessar o sistema no Render Código Fonte no GitHub

Descrição do Sistema

O sistema permite que os usuários comentem e visualizem, permitindo também que outros membros da comunidade manifestem interesse nos comentários e façam suas próprias publicações e deem suas opiniões.

Focado nas melhores práticas de TDD (Test-Driven Development) e Shift-Left, cada funcionalidade crítica foi desenvolvida com garantia prévia de conformidade, eliminando regressões desde o primeiro commit.
